Sum of two numbers

  • Input first number.
  • Input second number.
  • Add numbers.
  • Output sum.




Greatest among three

  • Input three numbers.
  • Compare first with second.
  • Compare larger with third.
  • Output largest number.




Odd or even

  • Input a number.
  • Divide number by two.
  • Check remainder.
  • If remainder is zero, output "even".
  • If remainder is not zero, output "odd".




Leap year or not

  • Input a year.
  • If year is divisible by 4, go to next step; otherwise, output "not a leap year".
  • If year is divisible by 100, go to next step; otherwise, output "leap year".
  • If year is divisible by 400, output "leap year"; otherwise, output "not a leap year".




Eligibility criteria to vote

  • Input age.
  • If age is 18 or older, output "eligible to vote".
  • If age is less than 18, output "not eligible to vote".




Valid date

  • Input day, month, and year.
  • Check if the month is valid (1-12).
  • Check if the day is valid for the given month and year.
  • Consider leap years when checking February.
  • If all checks pass, output "valid date".
  • If any check fails, output "invalid date".




Fibonacci series

  • Input the number of terms.
  • Initialize the first two terms as 0 and 1.
  • For each subsequent term, add the previous two terms.
  • Output the series up to the specified number of terms.




Factorial number

  • Input a non-negative integer.
  • Initialize the result as 1.
  • For each number from 1 to the input number, multiply the result by the current number.
  • Output the final result.




Prime number

  • Input a positive integer.
  • If the number is less than 2, output "not a prime number".
  • Check for factors from 2 to the square root of the number.
  • If any factor is found, output "not a prime number".
  • If no factors are found, output "prime number".




Perfect number

  • Input a positive integer.
  • Find all divisors of the number (excluding the number itself).
  • Sum the divisors.
  • If the sum equals the original number, output "perfect number".
  • If the sum does not equal the original number, output "not a perfect number".




Perfect number

  • Input a positive integer.
  • Find all divisors of the number (excluding the number itself).
  • Sum the divisors.
  • If the sum equals the original number, output "perfect number".
  • If the sum does not equal the original number, output "not a perfect number".




Armstrong number

  • Input a positive integer.
  • Calculate the number of digits (n) in the number.
  • For each digit, raise it to the power of n and sum the results.
  • If the sum equals the original number, output "Armstrong number".
  • If the sum does not equal the original number, output "not an Armstrong number".




Ascending order

  • Input an array of numbers.
  • Sort the array in ascending order using a sorting algorithm (e.g., bubble sort, quick sort).
  • Output the sorted array.




Matrix rotation

  • Input a 2D matrix.
  • Rotate the matrix 90 degrees clockwise.
  • Output the rotated matrix.




Palindrome

  • Input a string.
  • Reverse the string.
  • If the original and reversed strings are equal, output "palindrome".
  • If they are not equal, output "not a palindrome".




Reverse a string

  • Input a string.
  • Reverse the string.
  • Output the reversed string.




Binary search

  • Input a sorted array and a target value.
  • Set the left and right pointers.
  • Repeat until the target is found or the pointers cross.
  • If the target is found, output the index.
  • If the pointers cross, output "target not found".




Bubble sort

  • Input an array.
  • Repeat until the array is sorted.
  • Compare adjacent elements and swap if necessary.
  • Output the sorted array.




Merge sort

  • Input an array.
  • Divide the array into halves.
  • Recursively sort each half.
  • Merge the sorted halves.
  • Output the sorted array.




Quick sort

  • Input an array.
  • Choose a pivot element.
  • Partition the array around the pivot.
  • Recursively sort the partitions.
  • Output the sorted array.




Insertion sort

  • Input an array.
  • Repeat for each element from the second to the last.
  • Compare the current element with the previous elements and insert it in the correct position.
  • Output the sorted array.




Selection sort

  • Input an array.
  • Repeat for each element in the array.
  • Find the minimum element in the unsorted part of the array and swap it with the first unsorted element.
  • Output the sorted array.
